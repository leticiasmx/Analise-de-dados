# Visualização de Dados

## Descrição

Esta atividade faz parte de um desafio de análise de dados, onde foram aplicados conceitos de visualização de dados utilizando as bibliotecas **Pandas**, **numpy**, **Matplotlib** e **Seaborn** em Python.

Os desafios abordam diferentes tipos de gráficos e seus usos apropriados com base nos princípios de **Cleveland-McGill** (precisão perceptual) e **Gestalt** (percepção visual).


## Tecnologias Utilizadas

- Python 3.x
- Pandas
- Matplotlib
- Seaborn
- NumPy
